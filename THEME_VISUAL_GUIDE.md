## Accessibility Features

### All Themes Include:
✅ **High Contrast**: Text meets WCAG AA standards (4.5:1 minimum)
✅ **Focus Indicators**: Clear focus rings on interactive elements
✅ **Color Independence**: Information not conveyed by color alone
✅ **Readable Fonts**: Consistent typography across themes
✅ **Touch Targets**: Minimum 44x44px for mobile

### Contrast Ratios

#### Light Theme
- Background to Text: 15.8:1 (AAA)
- Primary to White: 4.5:1 (AA)
- Links to Background: 7.2:1 (AAA)

#### Dark Theme
- Background to Text: 14.2:1 (AAA)
- Primary to Background: 8.1:1 (AAA)
- Links to Background: 6.9:1 (AAA)

#### Multicolour Theme
- Background to Text: 13.5:1 (AAA)
- Primary to White: 4.6:1 (AA)
- Links to Background: 7.8:1 (AAA)

## User Preferences

### When to Use Each Theme

#### Light Theme
👍 Best for:
- Daytime use
- Bright environments
- Printing content
- Professional presentations
- Users with astigmatism

#### Dark Theme
👍 Best for:
- Nighttime use
- Low-light environments
- Reducing eye strain
- OLED battery saving
- Extended reading sessions

#### Multicolour Theme
👍 Best for:
- Creative work
- Engaging presentations
- Younger audiences
- Fun learning experiences
- Standing out from typical designs
