## 📖 How to Use

### Step 1: Browse Projects
1. Go to **Projects Explorer** page
2. See the "Unlimited Projects Available!" banner
3. Browse through available projects

### Step 2: Filter (Optional)
1. Use search box to find specific projects
2. Select category: Robotics or IoT
3. Select difficulty level
4. Projects update automatically

### Step 3: Load More
1. Scroll to bottom of page
2. Click **"Load More Projects"** button
3. 12 new projects appear
4. Repeat unlimited times!

### Step 4: View Project Details
1. Click **"View Details"** on any project
2. See complete project information
3. Read step-by-step instructions
4. Scroll to "Arduino/IoT Code" section

### Step 5: Copy Code
1. Hover over the code block
2. Click **"Copy Code"** button
3. See "Copied!" confirmation
4. Paste into Arduino IDE (Ctrl+V or Cmd+V)

### Step 6: Build Your Project
1. Read hardware connection details in code comments
2. Connect components as specified
3. Install required libraries (if mentioned)
4. Upload code to your board
5. Test and enjoy!
